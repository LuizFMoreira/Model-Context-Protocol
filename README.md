## 📝 Sobre este Trabalho

Este repositório contém o resumo crítico do artigo acima, elaborado como atividade avaliativa da disciplina de **Projeto de Software**.

O resumo aborda os seguintes tópicos extraídos do artigo:

- Contexto e motivação do Model Context Protocol (MCP)
- Arquitetura do sistema (Host, Client e Server)
- Ciclo de vida de um servidor MCP (criação, implantação, operação e manutenção)
- Panorama do ecossistema e adoção na indústria
- Ameaças de segurança identificadas (16 cenários em 4 categorias de atacantes)
- Recomendações dos autores para adoção segura
- Reflexão crítica do estudante
